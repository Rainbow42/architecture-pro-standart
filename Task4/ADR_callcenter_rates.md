### **Название задачи:**

Передача актуальных ставок по депозитам в кол-центр банка и партнёрский кол-центр (дополнение MVP)

### **Автор:**

Команда цифровой трансформации

### **Дата:**

13.06.2026

### **Функциональные требования**


| **№** | **Действующие лица или системы**         | **Use Case**             | **Описание**                                                                                                                                                |

### **Нефункциональные требования**


| **№** | **Требование**                                                   |
| ----- | ---------------------------------------------------------------- |
| NFR-1 | API ставок для кол-центра — только чтение, без доступа к заявкам |
| NFR-2 | Задержка обновления ставок в кол-центре — не более 15 минут      |
| NFR-3 | SFTP-выгрузка для партнёра — минимум 1 раз в день                |
| NFR-4 | Файл ставок — простой формат (CSV), понятный партнёру            |
| NFR-5 | Без API у партнёрского кол-центра — только SFTP                  |


### **Решение**

#### Архитектурное решение

Используем уже спроектированный **DAS** как единую точку публикации ставок:

- **REST API read-only** `/api/v1/deposit-rates` для CRM кол-центра банка;
- **Scheduled job Rate Export** — ежедневная генерация CSV и выкладка на SFTP.

Источник истины по ставкам — **модуль ставок в АБС (Oracle)**. DAS синхронизирует ставки из АБС и раздаёт их каналам.

#### Диаграммы

- C4 Context: `c4_context_callcenter.drawio`
- C4 Component: `c4_component_callcenter.drawio`

#### Компоненты


| Компонент                    | Назначение                    |
| ---------------------------- | ----------------------------- |
| Rate Management Module (АБС) | Хранение и изменение ставок   |
| Rate Sync Job (DAS)          | Синхронизация ставок из АБС   |
| Rates Read API (DAS)         | REST для кол-центра банка     |
| Rate Export Job (DAS)        | CSV → SFTP для партнёра       |
| CRM кол-центра               | UI оператора, вызов Rates API |
| SFTP-сервер банка            | Приём файлов для партнёра     |


### **Альтернативы**


| Альтернатива                           | Недостаток                          |
| -------------------------------------- | ----------------------------------- |
| Прямой доступ кол-центра к Oracle АБС  | Нагрузка на АБС, риски безопасности |
| Email со ставками (как сейчас с Excel) | Не автоматизировано, ошибки версий  |
| API для партнёрского кол-центра        | Партнёр не поддерживает API         |


### **Недостатки, ограничения, риски**

- Задержка до 15 минут между изменением ставки и отображением в кол-центре.
- Партнёр получает ставки только раз в день — возможна устаревшая информация.
- SFTP требует настройки безопасности (ключи, мониторинг).
- При падении DAS кол-центр не видит актуальные ставки — нужен fallback (кэш в CRM).
