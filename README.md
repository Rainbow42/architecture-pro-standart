# architecture-pro-standart

Проектная работа 3 спринта — цифровая трансформация банка «Стандарт».

## Структура репозитория

| Директория | Содержание |
|------------|------------|
| `tasks/` | Условия задания, шаблоны, Business Capability Map |
| `Task1/` | Карта IT-ландшафта и схема интеграции (draw.io) |
| `Task2/` | Таблица требований FURPS+ |
| `Task3/` | ADR и C4-диаграммы MVP депозитов |
| `Task4/` | ADR ставок в кол-центр, задачи по системам, RoadMap |
| `Task5/` | ADR и C4-диаграммы онлайн-кредитования |

## Ключевое архитектурное решение

- **DAS** (Deposit Application Service) — прослойка между каналами и АБС для депозитов.
- **CAS** (Credit Application Service) — прослойка для онлайн-кредитов и прямой интеграции с Кредитным конвейером.

Диаграммы открываются в [draw.io](https://app.diagrams.net/) или расширении Draw.io Integration для VS Code.
